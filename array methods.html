Array Methods In Javascript

  1. Push(): Push() is used to push/add item to the array at last position.
  2. Pop(): Pop() is used to remove item from the array from last position.
  3. unshift(): This function will add item at the 1st position in an array.
  4. shift(): Will remove item from the 1st position
  5. length: Finds the length of an Array.
  6. toString(): Will convert array into string format.
  7. join(): it is used to seperate the array items with some special characters during displaying on the screen
  8. concat(): Concat is used to merge 2 or more arrays into a single array.
  9. Sort(): Sort() will sort the array items in specific asscending order
  10. reverse(): Will sort the array items in descending order(reverse order)
  11. splice(): Splice method will add an item in to an array at a given position.
  12. Array.slice() returns selected array elements as a new array
  13. map() : The map() method creates a new array by performing a function on each array element.
  14. filter(): The filter() method creates a new array with array elements that passes a test.
  15. reduce(): The reduce() method runs a function on each array element to produce (reduce it to) a single value.
  16. indexOf(): The indexOf() method searches an array for an element value and returns its position.
  17. findIndex(): The findIndex() method returns the index of the first array element that passes a test function.
  18. keys(): The Array.keys() method returns an Array Iterator object with the keys of an array.
